TextArea-Expander
=================

	Tweeked and updated by dalemysz @ daleslab.com
	License: MIT
	To use: Include jQuery and this file on your page and include the class name expand in your textareas class attribute
	i.e. <textarea id="myText" class="someClass expand"></textarea>
	Alternatively if you are one of the granular people you can attach the plugin to your textarea with jQuery:
	$("#myText").autosize();
